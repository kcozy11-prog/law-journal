Google Calendar 연동 설정

Google Calendar API 직접 연동

1
Google Cloud Console에서 프로젝트 생성 및 API 활성화
아래 링크에서 새 프로젝트를 만들고, Google Calendar API를 검색해 활성화합니다.
(이미 프로젝트가 있다면 그대로 사용하셔도 됩니다)
🔗 Google Calendar API 활성화 페이지 열기
2
OAuth 2.0 클라이언트 ID 발급
사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID
애플리케이션 유형: 웹 애플리케이션
승인된 JavaScript 출처에 이 파일을 열 주소 추가 (로컬: http://localhost 또는 실제 URL)

⚠️ OAuth 동의 화면도 먼저 설정해야 합니다 (테스트 사용자에 본인 이메일 추가)
🔗 사용자 인증 정보 페이지 열기
3
Client ID 입력
발급받은 클라이언트 ID를 붙여넣으세요. xxxxxxxxx.apps.googleusercontent.com 형식입니다.
✅ 저장됨
4
Google 계정으로 로그인
Client ID 저장 후 아래 버튼으로 Google 계정에 로그인합니다. Calendar 읽기 권한만 요청합니다.
5
연동 테스트
로그인 완료 후 오늘 일정을 불러와 정상 작동을 확인합니다.
