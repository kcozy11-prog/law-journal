# Snoopy Valentine — 영어 학습지

피너츠 발렌타인 에피소드 *Hearts For My Sweet Baboo* 를 활용한 영어 빈칸 채우기 학습 앱입니다.

## 기능

- **장면 1~4** 의 영어 대본 빈칸 채우기
- **단어 상자**(Word Box) — 정답 단어가 섞여서 제시됨
- **정답 보기 / 숨기기** 토글 버튼
- 입력 시 실시간 정답/오답 표시
- 한글 해석 동시 노출, 학습 팁 사이드바

## 실행

```bash
cd snoopy-valentine
npm install
npm run dev
```

빌드:

```bash
npm run build
npm run preview
```

## 기술 스택

- React 18
- Vite 5
- 순수 CSS (Pretendard / Apple SD Gothic Neo)
